# לימוד הא'-ב' לילדה

אפליקציה סטטית צבעונית ללימוד האותיות בעברית, שמתאימה לילדה בת 5 שמתחילה להכיר את הא'-ב'.

## מה יש באפליקציה

- "הכרות עם אותיות" עם כרטיסים של כל אות.
- כפתור קריאה לכל אות ודיבור עברי נכון.
- כפתור "למדתי את האות" לפני שמשחקים.
- שני משחקים חכמים שמתחילים רק אחרי שהאות הוכרה.
- סטטיסטיקה יומית של ניסיונות והצלחות בכל משחק.
- ייצוא וייבוא של התקדמות כדי להעביר למכשיר אחר.

## איך לשים על GitHub

1. פתחי תיקייה חדשה ב־GitHub עם שם שמתאים, למשל `alefbet-learning`.
2. העתקי את כל הקבצים בספרייה זו וודאי שהם נמצאים במאגר.
3. אם את רוצה להמשיך לערוך ממכשיר אחר, פשוט תשכפלי את המאגר שלך למכשיר אחר או לפתוח אותו ב־GitHub.
4. האפליקציה עובדת סטטית על כל שרת HTTP, וגם על GitHub Pages.

## איך לחבר Supabase

1. היכנסי ל־https://supabase.com וצרי פרויקט חדש.
2. בעמוד ה־Settings קחי את `URL` ו־`anon key` עבור הפרויקט.
3. הוסיפי טבלה בשם `alpha_bet_learning` עם השדות:
   - `date` (type: `text` או `date`)
   - `game` (type: `text`)
   - `plays` (type: `integer`)
   - `wins` (type: `integer`)
4. ב־index.html, וודאי שקיים הסקריפט:
   - `<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2/dist/supabase.min.js"></script>`
5. ב־טאב `סטטיסטיקה` הכניסי את ה־URL וה־anon key ולחצי `התחבר ל-Supabase`.
6. כעת כל משחק יוצג גם ב־Supabase במידה והחיבור מצליח.

## איך להפעיל

- פתחי את `index.html` בדפדפן.
- או תחברי עם שרת סטטי כמו GitHub Pages או `python -m http.server`.

## קבצים

- `index.html` - ממשק המשתמש והמבנה.
- `style.css` - עיצוב צבעוני ונעים.
- `app.js` - לוגיקת המשחקים, שמירת מצב והסטטיסטיקה.

## הערות

- אין כאן תלות בשירות ענן חיצוני.
- כל הנתונים נשמרים בדפדפן באמצעות `localStorage`.
- ניתן לייצא קובץ JSON אם רוצים לשמור גיבוי או להעביר למכשיר אחר.
